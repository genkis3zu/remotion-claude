# BGMガイド

`public/bgm/` に格納されているBGMファイルの分類と使用ガイドです。
動画の雰囲気に合わせて適切なBGMを選択してください。

---

## 実装例

`src/data/script.ts`

```typescript
// 日常的な解説動画
export const bgmConfig: BGMConfig = {
  src: "Morning.mp3",
  volume: 0.2,
  loop: true,
};

// クイズ動画
export const bgmConfig: BGMConfig = {
  src: "なんでしょう？.mp3",
  volume: 0.3,
  loop: true,
};

// 感動的なエンディング
export const bgmConfig: BGMConfig = {
  src: "Recollections.mp3",
  volume: 0.4,
  loop: false,
};
```
