## 📝 使い方

`src/data/script.ts` などで指定します。
パスは `public/` からの相対パスで指定してください。

```typescript
// 驚き
{
  text: "ええええ！？",
  emotion: "surprised",
  se: {
    src: "se/anime/驚く.mp3",
    volume: 1.0
  }
}

// クイズ正解
{
  text: "正解なのだ！",
  emotion: "happy",
  se: {
    src: "se/quiz/クイズ正解1.mp3",
    volume: 0.8
  }
}

// ボタンクリック（解説など）
{
  text: "このボタンを押すと...",
  se: {
    src: "se/button-push/決定ボタンを押す22.mp3",
    volume: 0.5
  }
}
```
